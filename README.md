### Linux環境

フォントのインストール<br/>
[0xProto](https://github.com/0xType/0xProto)<br/>

SQLite3のライブラリインストール (Debian / Ubuntu)<br/>
```Shell
$ sudo apt install libsqlite3-dev -y
```

アーカイブファイルの解凍とKakeiBonの実行
```Shell
$ mkdir -p /path/to
$ cd /path/to
$ tar xvf ./KakeiBon-linux-x64-[バージョン].tar.xz
$ ./KakeiBon
```

### Windows環境

１．Releaseページの'sqlite-dll-win-x64-[バージョン].zip'ファイルをダウンロードの後、<br/>
　　展開してDLLファイルをC:\Windows\system32\などのフォルダに格納するか、<br/>
　　任意のフォルダに格納してパスを通してください。<br/>
２．同じくReleaseページより'KakeiBon-win-x64-[バージョン].zip'ファイルをダウンロード<br/>
　　して、展開したEXEファイルを実行してください。<br/>

### Linux / Windows 共通

KakeiBon実行時にユーザデータを格納するSQLite3(データベース)の<br/>
ファイルを作成する(初回のみ)ため、メインウィンドウ(メニュー画面)が<br/>
開いたあと、管理者ユーザを作成するウィンドウが表示されるまで<br/>
少しお待ちください。<br/>
おおよそ数秒〜10秒程度で管理者登録用のウィンドウが開きます。<br/>

全体的な流れとしては、<br/>
１．KakeiBonの起動(メインメニュー画面が開く)<br/>
２．自動で管理者登録用ウィンドウが開く<br/>
３．管理者を登録<br/>
４．メニュー画面からログイン<br/>
５．ユーザ管理画面から通常使用するユーザを登録<br/>
６．メニュー画面からログアウト(管理者ユーザ)<br/>
７．メニュー画面からログイン(通常使用するユーザ)<br/>
８．[任意] 入出金を登録する際に使用する費目(例:食費、交通費など)の<br/>
　　カスタマイズをメニュー画面の費目管理から行えます。<br/>
９．メインメニューの入出金登録より、実際の入出金を登録・編集<br/>
　　していきます。入力項目の横もしくは上にボタンがある場合、<br/>
　　必要に応じてボタンを押下し、サブウィンドウでデータを<br/>
　　登録してください。そうすることで呼び出し元画面で<br/>
　　登録したデータを呼び出せるようになります。<br/>

お買い物などの商品データなど、ある程度登録できるまでは手間が<br/>
かかりますが、データが蓄積できてくると、そのデータの選択で<br/>
入力を簡素化して、入出金の登録時間を短縮していけます。<br/>
また、ボタンはラベル表記にあるように、ショートカットキーを用意<br/>
しています。慣れると、かなりの速度で入出金を登録できるように<br/>
なるでしょう。<br/>
