### Linux環境

SQLite3のライブラリインストール
```Shell
$ sudo apt install libsqlite3-dev -y
```

アーカイブファイルの解凍とKakeiBonの実行
```Shell
$ tar xvf ./KakeiBon-linux-x64-[version].tar.xz
$ ./KakeiBon
```

KakeiBon実行時にユーザデータを格納するSQLite3(データベース)の
ファイルを作成するため、メインウィンドウ(メニュー画面)が
開いたあと、管理者ユーザを作成するウィンドウが表示されるまで
少しお待ちください。
おおよそ数秒〜10秒程度で管理者登録用のウィンドウが開きます。

全体的な流れとしては、
１．KakeiBonの起動(メインメニュー画面が開く)
２．自動で管理者登録用ウィンドウが開く
３．管理者を登録
４．メニュー画面からログイン
５．ユーザ管理画面から通常使用するユーザを登録
６．メニュー画面からログアウト(管理者ユーザ)
７．メニュー画面からログイン(通常使用するユーザ)
８．[任意] 入出金を登録する際に使用する費目(例:食費、交通費など)の
　　カスタマイズをメニュー画面の費目管理から行えます。
９．メインメニューの入出金登録より、実際の入出金を登録・編集
　　していきます。入力項目の横もしくは上にボタンがある場合、
　　必要に応じてボタンを押下し、サブウィンドウでデータを
　　登録してください。そうすることで呼び出し元画面で
　　登録したデータを呼び出せるようになります。

お買い物などの商品データなど、ある程度登録できるまでは手間が
かかりますが、データが蓄積できてくると、そのデータの選択で
入力を簡素化して、入出金の登録時間を短縮していけます。
また、ボタンはラベル表記にあるように、ショートカットキーを用意
しています。慣れると、かなりの速度で入出金を登録できるように
なるでしょう。
